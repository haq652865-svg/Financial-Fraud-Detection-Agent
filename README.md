# Financial Fraud Detection Agent

An AI-powered conversational agent designed to identify red flags 
in financial transactions using pattern recognition.

## Built With
- Flowise (Agent Framework)
- Mistral AI (LLM Backend)

## Features
- Detects suspicious financial patterns
- Conversational interface for query-based analysis
- Applicable for financial investigation workflows

## How to Use
1. Import the exported flow JSON into Flowise
2. Connect your Mistral AI API key
3. Run the agent and input transaction details
